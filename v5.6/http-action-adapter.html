# Http 操作适配器

HttpActionAdapter (opens new window) 是一个抽象，用于 应用 pac4j HTTP 操作（=HttpAction (opens new window)），这些动作由 HTTP 请求/响应 = web 上下文 上的逻辑返回。

它的实现因 pac4j 的实现而异。

例如，有用于 JEE 应用程序的 JEEHttpActionAdapter (opens new window)、用于 Play应用程序的 PlayHttpActionAdapter 等。
